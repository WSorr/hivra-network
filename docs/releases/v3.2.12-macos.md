# Hivra v3.2.12 for macOS

## Download

Recommended package:

- `Hivra-v3.2.12-macos.dmg`

Alternative package:

- `hivra_app-v3.2.12-macos-release.zip`

## Install

1. Download `Hivra-v3.2.12-macos.dmg`.
2. Open Terminal and run:

```bash
xattr -dr com.apple.quarantine ~/Downloads/Hivra-v3.2.12-macos.dmg
```

3. Open the `.dmg` file.
4. Drag `Hivra.app` into `Applications`.
5. Run:

```bash
xattr -dr com.apple.quarantine /Applications/Hivra.app
```

6. Open `Applications` and launch `Hivra`.

## Important

This build is currently not Apple signed or notarized. On recent macOS versions, Gatekeeper may block the downloaded disk image or the installed app unless the quarantine attribute is removed first.

## If macOS still blocks the app

1. Open `Applications`.
2. Right-click `Hivra.app`.
3. Select `Open`.
4. Click `Open` again in the confirmation dialog.

If needed:

1. Try to open `Hivra.app` once.
2. Open `System Settings -> Privacy & Security`.
3. Find the blocked app warning near the bottom.
4. Click `Open Anyway`.

## If macOS says the app is damaged

This is usually a Gatekeeper quarantine issue, not actual file corruption.

Run this in Terminal:

```bash
xattr -dr com.apple.quarantine /Applications/Hivra.app
```

Then try opening the app again.
